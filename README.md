# Demux_1X8
Implementation of a 1 to 8 demultiplexer using two 1 to 4 demultiplexers and one 1 to 2 demultiplexer in VHDL.

## Built With
-Quartus Prime Version 20.1.1 Build 720 11/11/2020 SJ Lite Edition<br />
